# Data Cleaning & Preparation – Project 1

This repository contains my submission for Project 1 of the DecodeLabs Data Analytics Internship Program.

## Overview
The project focuses on cleaning and preparing a raw e-commerce dataset using Excel and Julius AI to improve data quality and prepare the data for analysis.

## Objectives
- Handle missing values
- Remove duplicate records
- Correct inconsistent data formats
- Prepare the dataset for analysis

## Tools Used
- Microsoft Excel
- Julius AI
